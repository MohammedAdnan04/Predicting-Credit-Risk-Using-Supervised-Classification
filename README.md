# Data Science and Machine Learning — COMP4030

## Overview
This repository contains coursework and applied exercises completed for COMP4030 (Data Science and Machine Learning) at the University of Nottingham. The project covers the full DSML workflow from raw data through to model evaluation and interpretation.

## Topics Covered
- Data wrangling and cleaning
- Exploratory Data Analysis (EDA)
- Unsupervised learning — clustering
- Supervised learning — regression and classification
- Experiment design and model evaluation
- Time series forecasting

## Tech Stack
- Python
- Scikit-learn
- Pandas
- Matplotlib / Seaborn
- Jupyter Notebook / Google Colab

## How to Run
1. Clone this repository
2. Open the relevant `.ipynb` notebook in Jupyter or Google Colab
3. Install dependencies if needed:
```bash
pip install scikit-learn pandas matplotlib seaborn
```
4. Run cells sequentially
